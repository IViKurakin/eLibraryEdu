# Доклад по HTML-шаблонам проекта eLibrary (Django)

## Обзор проекта
Проект eLibrary реализован на Django и включает 9 основных HTML-шаблонов, обеспечивающих функциональность электронной библиотеки.

## Структура шаблонов

### 1. **base.html** - Базовый шаблон
```html
<!-- Примерная структура -->
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>{% block title %}eLibrary{% endblock %}</title>
    <!-- Общие стили и скрипты -->
</head>
<body>
    <header>Навигация</header>
    <main>{% block content %}{% endblock %}</main>
    <footer>Футер</footer>
</body>
</html>
```
**Назначение**: Основной каркас для всех страниц
**Наследуют**: Все остальные шаблоны

### 2. **home.html** - Главная страница
**Наследует**: `base.html`
**Функционал**:
- Приветствие пользователей
- Краткая информация о библиотеке
- Быстрый доступ к основным разделам
- Статистика (количество книг, пользователей и т.д.)

### 3. **login.html** - Страница входа
**Наследует**: `base.html`
**Элементы**:
- Форма входа (username, password)
- Ссылка на регистрацию
- Валидация данных
- Обработка ошибок аутентификации

### 4. **register.html** - Регистрация пользователя
**Наследует**: `base.html`
**Поля формы**:
- Имя пользователя
- Email
- Пароль (с подтверждением)
- Дополнительная информация (опционально)

### 5. **explore.html** - Каталог книг
**Наследует**: `base.html`
**Функции**:
- Поиск книг по различным критериям
- Фильтрация (по жанру, автору, дате)
- Сортировка
- Пагинация результатов
- Превью книг

### 6. **viewBook.html** - Просмотр книги
**Наследует**: `base.html`
**Содержит**:
- Полная информация о книге
- Обложка
- Аннотация
- Сведения об авторе
- Оценки и отзывы
- Кнопки действий (редактировать/удалить для админов)

### 7. **addBook.html** - Добавление новой книги
**Наследует**: `base.html`
**Поля формы**:
- Название книги
- Автор(ы)
- ISBN
- Жанр
- Год издания
- Описание
- Загрузка обложки
- Файл книги (опционально)

### 8. **editBook.html** - Редактирование книги
**Наследует**: `base.html`
**Особенности**:
- Предзаполненные поля текущими данными
- Валидация изменений
- История изменений
- Подтверждение сохранения

### 9. **contri.html** - Страница вкладчиков/авторов
**Наследует**: `base.html`
**Содержание**:
- Информация об авторах
- Статистика вкладов
- Список добавленных книг
- Рейтинги и достижения

## Архитектурные особенности

```
templates
├── base.html
├── home.html
├── login.html
├── register.html
├── explore.html
├── viewBook.html
├── addBook.html
├── editBook.html
└── contri.html
```

### Блоки Django-шаблонов
Каждый шаблон использует:
- `{% block title %}` - заголовок страницы
- `{% block content %}` - основное содержание
- `{% block styles %}` - дополнительные стили
- `{% block scripts %}` - специфичные скрипты

## Стилевые и скриптовые зависимости

### Общие ресурсы (в base.html):
1. CSS-фреймворк (Bootstrap/Tailwind)
2. Иконки (FontAwesome)
3. Основные JavaScript библиотеки
4. Пользовательские стили (styles.css)

### Страничные ресурсы:
- **explore.html**: Скрипты для фильтрации и поиска
- **addBook.html/editBook.html**: Валидация форм, загрузка файлов
- **viewBook.html**: Рейтинговая система, комментарии

## Формы Django

### Пример формы (addBook.html):
```html
<form method="POST" enctype="multipart/form-data">
    {% csrf_token %}
    {{ form.as_p }}
    <button type="submit">Добавить книгу</button>
</form>
```

### Типы форм в проекте:
1. `AuthenticationForm` - login.html
2. `UserCreationForm` - register.html
3. `BookForm` - addBook.html, editBook.html
4. `SearchForm` - explore.html

## Безопасность

### Реализованные меры:
1. CSRF-токены во всех формах
2. Проверка прав доступа:
   - Только авторизованные пользователи: addBook.html, editBook.html
   - Публичный доступ: home.html, explore.html, viewBook.html
3. Валидация файловых загрузок
4. Экранирование пользовательского контента

## Адаптивность

Все шаблоны разработаны с учетом:
- Мобильных устройств
- Планшетов
- Десктопных браузеров
- Различных разрешений экрана
