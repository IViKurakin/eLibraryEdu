## Что протестировано (всё основное)
- Регистрация нового пользователя  
- Регистрация уже существующего email (сообщение об ошибке)  
- Логин / неверный пароль (сообщение об ошибке)  
- Логаут  
- Добавление книги с PDF-файлом  
- Редактирование своей книги (в том числе смена PDF)  
- Удаление своей книги  
- Просмотр детальной страницы книги (замена \n → <br/> работает)  
- Страница Explore — корректная фильтрация по категориям Education / Fiction / Science  
- Личный кабинет (contri) — показывает только свои книги  
- Защита всех @login_required маршрутов  
- Валидация формы добавления/редактирования книги  

## Как запустить тесты локально
```bash
pip install -r requirements.txt
coverage run -m pytest -ra
coverage report -m          # в терминале
coverage html               # красивый отчёт в htmlcov/index.html
coverage-badge -o coverage.svg
```
